## Figma vs Sketch vs Penpot: Design Tool Comparison

Three design tools for different workflows and budgets.

### At a Glance

| | Figma | Sketch | Penpot |
|--|-------|--------|--------|
| Platform | Web + desktop | macOS only | Web |
| Collaboration | Real-time | Via plugins | Real-time |
| Free tier | Yes (3 files) | No (30-day trial) | Yes (unlimited) |
| Open source | No | No | Yes |
| Dev handoff | Built-in | Via plugins | Built-in |

### Figma — The Industry Standard

Web-based, real-time collaboration, and the best prototyping tools. The free tier works for most individuals. Dev mode shows CSS directly.

### Sketch — The Mac Classic

macOS-only vector design tool. Massive plugin ecosystem. One-time purchase option available. Best for solo designers on Mac.

### Penpot — The Open Source Challenger

Fully open source, web-based, real-time collaboration. Supports SVG natively. Best for teams that want Figma features without Figma pricing.
